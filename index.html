⚠️ Réinitialisation du stock à zéro

Supprime tous les mouvements (entrées, sorties, transferts) et remet les quantités initiales à 0.
Les produits, fournisseurs et commandes sont conservés.
⚠️ Cette action est irréversible.

📋 Conseil : Exportez d'abord un rapport PDF (inventaire + mouvements) avant de réinitialiser.

🚨 Cette action est totalement irréversible.

🛑 Utiliser uniquement pour repartir sur une base vierge. Toutes les données opérationnelles seront perdues.
